##Run  component:
> make run-docker

The following environment variables can be set in _docker-compose.yml_:
- RASA_API_URL   = http://localhost:5005/webhooks/rest/webhook
- TTS_API_URL    = http://localhost:6901
- TTS_API_USERNAME
- TTS_API_PASSWORD
- STT_API_URL
- SRV_PORT
- SRV_HOST
- APP_LOG_LEVEL
